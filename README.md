📊 **Credit Risk Analysis Dashboard | Power BI Project**

I’m excited to share my **Credit Risk Analysis Dashboard**, built to analyze customer credit behavior and identify potential default risks using interactive data visualization.

🔢 **Key Metrics**
• **Total Customers:** 10,000
• **Average Credit Score:** 625
• **Default Rate:** 13%
• **High-Risk Customers:** 4,000

📈 **Dashboard Insights**
✔️ Analyzed monthly **default trends** to identify risk patterns over time.
✔️ Segmented customers into **High, Medium, and Low Risk groups** based on credit scores.
✔️ Explored relationships between **employment type, income levels, and default status**.
✔️ Compared **loan decisions with default outcomes** to evaluate lending effectiveness.
